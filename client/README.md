# Auto Stitch & Zoom Client

React + TypeScript + Vite frontend for the Auto Stitch & Zoom video app.

## Scripts

- `npm run dev` starts the local Vite dev server
- `npm run build` type-checks and creates a production build
- `npm run typecheck` runs the client TypeScript checks
- `npm run lint` runs the client ESLint checks

## Local development

The dev server proxies `/api` and `/uploads` to `http://localhost:3001`, so run the backend at the same time.
